# Dispensa di Reti Wireless e Mobili

Appunti del corso di _[Reti Wireless e Mobili](https://www.unimi.it/it/corsi/insegnamenti-dei-corsi-di-laurea/2026/reti-wireless-e-mobili-0)_ (a.a. 2025/26), tenuto dal Prof. _Christian Quadri_, Corso di Laurea in Informatica Magistrale, Università degli Studi di Milano.

Realizzati da [Luca Corradini](https://github.com/LucaCorra02), [Andrea Galliano](https://github.com/gallja), con il contributo di [altri collaboratori](https://github.com/LucaCorra02/Reti-Wireless-Mobili/graphs/contributors).

Questi appunti sono open source: [github.com/LucaCorra02/Reti-Wireless-Mobili](https://github.com/LucaCorra02/Reti-Wireless-Mobili) con licenza [CC-BY-4.0](https://creativecommons.org/licenses/by/4.0/).

Le contribuzioni e correzioni sono ben accette attraverso Issues o Pull Requests.

> [!CAUTION]
> Controllare soprattutto Issues e commenti `TODO`

## PDF Compilato

Il documento è scritto in [Typst](https://typst.app/) _(che è meglio di LaTeX)_.

- Un PDF compilato è generato automaticamente ad ogni push su `main` e disponibile nelle [release](https://github.com/your-username/your-repo/releases).
- Oppure è possibile compilare localmente con il comando: `typst compile main.typ`

## Workflow

Il push su `main` (e quindi la generazione del PDF) è protetto:

- è necessario creare una pull request per proporre modifiche
- è richiesta una review da parte di un collaboratore per il merge
- viene generato automaticamente un PDF temporaneo per ogni pull request
